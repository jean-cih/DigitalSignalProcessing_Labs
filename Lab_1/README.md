# Fundamentals of signal generation and analysis

Данноя работа посвящено фундаментальным принципам формирования и анализа сигналов. Включает синтез дискретных гармонических сигналов с заданными параметрами (амплитуда, частота, фаза), исследование их дискретизированного представления, проведение спектрального анализа с использованием различных оконных функций (например, Хэмминга) и визуализацию как временных, так и частотных характеристик сигналов. Также рассматриваются концепции двухстороннего и одностороннего амплитудного спектра.
