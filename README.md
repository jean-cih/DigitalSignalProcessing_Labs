# DigitalSignalProcessing_Labs

Этот репозиторий содержит коллекцию лабораторных работ по цифровой обработке сигналов и радиотехники. Работы охватывают ключевые концепции формирования, анализа, квантования, модуляции сигналов, генерации псевдослучайных чисел и цифрового спектрального анализа, предназначенные для получения практического опыта работы с основными методами обработки сигналов.
