# Modulated signals

Работа охватывает создание, анализ и демодуляцию различных типов модулированных сигналов. Включает синтез амплитудно-модулированных (АМ) и частотно-модулированных (ЧМ) сигналов, исследование синхронной амплитудной демодуляции, а также работу с сигналами DTMF (Dual-Tone Multi-Frequency). Расширенные задачи включают оценку нелинейных и интермодуляционных искажений в демодулированных сигналах.
