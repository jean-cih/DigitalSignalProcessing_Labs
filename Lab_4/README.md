# Basics of random number generation on a computer

Работа посвящена основам генерации псевдослучайных чисел на ЭВМ. Включает реализацию алгоритмов формирования последовательностей с равномерным распределением, расчет и анализ их автокорреляционных функций, построение гистограмм. Также рассматриваются методы получения последовательностей с нормальным распределением и распределением Рэлея.
