# Signal quantization

Работа фокусируется на изучении принципов квантования сигналов. Включает реализацию модели квантователя, анализ влияния разрядности квантователя на дискретный и цифровой сигналы, расчет мощности шума квантования и оценку отношения сигнал/шум квантования (СПМ). Также рассматривается моделирование аналого-цифрового преобразования (АЦП) с использованием простейших методов.
